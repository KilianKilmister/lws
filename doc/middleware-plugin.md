## middleware-plugin
MiddlewareStack plugin API.

**Example**  
```js
class Greeter {
  middleware (options = {}) {
    return function (ctx, next) {
      ctx.body = `Hello Mr ${options.surname}`
      next()
    }
  }

  optionDefinitions () {
    return [
      { name: 'surname', description: 'Your family name.' }
    ]
  }
}

module.exports = Greeter
```
